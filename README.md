# 頭竹苗福利社 GitHub Pages 多頁式官網

本版為多頁式正式官網，風格方向為簡約、乾淨、專業、清楚 CTA。

## 頁面
- index.html：首頁
- card.html：免費福利卡
- shops.html：特約商店查詢
- merchant.html：特約商專區（店家加入 + 資料更新）
- company.html：企業合作
- about.html：關於我們
- news.html：最新消息

已取消 update.html，所有資料更新入口都導向 merchant.html#update。

## 資料
- data/shops.json：特約商資料
- assets/img/logo/：特約商 Logo
- assets/img/logo/default-logo.svg：預設 Logo

## GitHub Pages
將整包檔案放到 repository 根目錄，Settings → Pages → Deploy from branch → main / root。
