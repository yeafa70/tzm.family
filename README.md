# 頭竹苗福利社｜圖三乾淨重寫版 v3

這版不是在舊版 CSS 上補丁，而是重新整理一份乾淨的圖三風格 CSS。

## 本版確認
- 不再沿用最早橘色版本
- 不再使用舊版橘色變數
- 不再使用舊版橘色碼
- 不再使用膠囊圓角
- 不再使用圓形卡片圓角
- 全站改為圖三色系：
  - 深茄紫 `#4B2E4F`
  - 鼠尾草綠 `#7D8F72`
  - 陶土玫瑰 `#C68873`
  - 暖石灰 / 暖白

## 本版視覺原則
- 方形為主
- 只保留 6px～8px 小圓角
- 不做膠囊按鈕
- 不做可愛大圓角
- 保留圖三的首頁區塊結構：Hero、分類 icon、店家卡、三大服務入口
